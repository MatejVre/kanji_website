ABOUT

The page let's you browse all of the JLPT kanji based on the level. If you wish to learn more about the JLPT, you can read about it on the official website.

Currently limited to JLPT kanji only -> other jouyou kanji to be added in the future.

SEARCH

The site allows search in three different ways:

Kanji: Searches based on the inputted kanji. To search for multiple kanji, simply write them all in the search box, no spaces or commas necessary.

Reading: Searched based on the inputted reading. It is important to note, that the results differ based on whether the input is in hiragana or katakana.
If searching for a kunyomi, use hiragana and if searching for onyomi, use katakana.
For example searching for こころ will return 心 in the result, where as ココロ will not.

Grade: Searches based on the inputted grade level. Supports number from 1 - 7.
